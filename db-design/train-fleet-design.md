# 車隊主檔定義

本文件固定 ERP 使用的車號主檔。坑位、工單、設備履歷、領料到車上，都應引用這份車號主檔。

## 場站代碼

| 代碼 | 場站 |
| --- | --- |
| `D` | 淡海 |
| `K` | 安坑 |

## 淡海列車

淡海目前 18 台：

```text
101-115、117、118、119
```

其中：

| 現行車號 | 原/對應車號 |
| --- | --- |
| 117 | 215 |
| 118 | 214 |
| 119 | 213 |

注意：依目前定義，淡海沒有 `116車`。

## 安坑列車

安坑目前 12 台：

```text
201-212
```

## 資料庫欄位

`train` 主檔保存：

| 欄位 | 用途 |
| --- | --- |
| `train_no` | 顯示車號，例如 `101車` |
| `fleet_name` | 路線/車隊名稱，例如 `淡海輕軌` |
| `site_code` | `D` 或 `K` |
| `line_name` | `淡海` 或 `安坑` |
| `former_train_no` | 原車號或對應車號，例如 `215車` |
| `display_order` | 排序用數字 |
| `is_active` | 是否仍納入日常作業 |

## 坑位展開規則

`周轉建0304.xlsx` 目前的 Location_ID 使用：

```text
50-D-TS101-M1-...
```

這代表它現在是淡海 `101車` 的坑位模板。

若要展開淡海全車隊，應替換 `TS101` 為：

```text
TS101, TS102, TS103, TS104, TS105,
TS106, TS107, TS108, TS109, TS110,
TS111, TS112, TS113, TS114, TS115,
TS117, TS118, TS119
```

若安坑使用相同坑位結構，才可另外展開：

```text
50-K-TS201-M1-...
```

到：

```text
TS201-TS212
```

如果安坑設備配置不同，安坑應另外提供坑位模板，不要直接套淡海模板。
